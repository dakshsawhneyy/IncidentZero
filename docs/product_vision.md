# Incident Zero

#### Vision: 
A simulator for Site Reliability Engineers where engineers solve realistic production incidents and receive feedback on how they investigated them.

I have faced this myself, that SRE freshers
don't get any platfrom to actually learn production incidents

So keeping that in mind, i built a platform which i will be using myself
for learnign and getting better in production reliability betterments


# How it's different from other Incident Labs
Other labs focus more on learning of tools and usually target scenarios before tool installation

Incident Zero is different:
It focuses on investigating production outages after systems are already running, rather than teaching how to install or configure the tools.


# How its special

```
Pager goes off.
↓
Open Grafana
↓
Open Prometheus
↓
Look at Logs
↓
Check Events
↓
Terminal
↓
Wrong hypothesis
↓
Second hypothesis
↓

↓

```
